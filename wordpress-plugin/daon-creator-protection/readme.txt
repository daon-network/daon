## Installation

### Automatic Installation

1. Go to your WordPress admin area
2. Navigate to Plugins > Add New
3. Search for "DAON Creator Protection"
4. Click "Install Now" and then "Activate"
5. Go to Settings > DAON Protection to configure

### Manual Installation

1. Download the plugin zip file
2. Go to Plugins > Add New > Upload Plugin
3. Choose the zip file and click "Install Now"
4. Activate the plugin
5. Configure settings in Settings > DAON Protection

### First Time Setup

1. **Enable Auto-Protection** - Automatically protect new content
2. **Choose Default License** - Liberation License recommended
3. **Select Post Types** - Which content to protect (posts, pages, etc.)
4. **Set Word Minimum** - Only protect substantial content
5. **Enable Protection Notice** - Show visitors your content is protected

== Frequently Asked Questions ==

= Is this really free? =

Yes! DAON Creator Protection is completely free with no limits. We believe creator protection should be accessible to everyone.

= Does this slow down my website? =

No. Protection happens in the background after your content is published. Visitors see no performance impact.

= What if DAON goes offline? =

Your content remains protected. The blockchain records are permanent and distributed across multiple validators worldwide.

= Can I protect existing content? =

Yes! You can manually protect existing posts from the edit screen, or use the bulk protection tool in the admin area.

= What's the difference between license types? =

- **Liberation License**: Blocks corporate AI training, allows personal/educational use
- **Creative Commons BY-NC**: Requires attribution, allows non-commercial use
- **Creative Commons BY-NC-SA**: Like BY-NC but requires derivative works use same license
- **All Rights Reserved**: Traditional full copyright protection

= Do I need a crypto wallet? =

No! The plugin handles all blockchain interactions automatically. No crypto knowledge required.

= Is my content stored on the blockchain? =

No. Only a cryptographic fingerprint (hash) of your content is stored. Your actual content stays on your WordPress site.

= Can I turn off protection notices? =

Yes. You can disable the protection notice display in Settings > DAON Protection.

= Does this work with my theme? =

Yes. DAON protection works with any properly coded WordPress theme.

= What about privacy/GDPR? =

DAON is hosted in Europe with full GDPR compliance. No personal data is stored without consent.

== Privacy Policy ==

DAON Creator Protection respects your privacy:

- **Content Hashing**: We generate a cryptographic hash of your content for blockchain storage
- **No Content Storage**: Your actual content is never stored on our servers
- **Metadata Collection**: Basic metadata (title, author, publish date) for protection records
- **EU Hosting**: All data processed in GDPR-compliant European data centers
- **No Tracking**: We don't track your website visitors or personal behavior
- **Opt-In Only**: Protection only happens when you choose to enable it
